近日，不少明星为纪念反法西斯战争胜利70周年，纷纷晒出敬礼照或者军装照，向抗战老兵致敬。事实上，娱乐圈里当过军人或出生军人家庭的明星不在少数。他们或多或少都经历过军旅生涯的锻炼，练就了顽强拼搏的军人性格，为日后进入娱乐圈打下坚实的基础。</p>
孙俪、刘涛都曾是文艺兵</p>
5岁时孙俪便进入少年宫学习舞蹈，初中毕业，上海警卫区文工团招收新兵，孙俪幸运地当上了一名文艺兵。董洁 10岁由战士歌舞团在大连特招入伍，后被送入北京的解放军艺术学院学了5年舞蹈。1995年底正式到战士歌舞团做舞蹈演员，后来因为和谢霆锋 在春晚的合作而被张艺谋 相中主演《幸福时光》成名。歌唱组合“凤凰传奇”2009年进入二炮文工团，那首红遍大江南北的《中国我爱你》就是凤凰传奇特意为2009年建国60周年大庆而献声的歌曲。</p>
演员刘涛1993年被选入南京军区集团军文工团，在那里当了3年的文艺兵；梅婷也曾有过8年的军旅生涯；而在年轻人中颇受欢迎的“小鲜肉”杨洋，同样毕业于解放军艺术学院舞蹈系，是一名文艺兵。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 资料图：韩红。 中新社发 陈文 摄
韩红、闫妮目前仍是军人</p>
也有不少明星目前仍是现役军人。除了知名歌手韩红之外，曾在《武林外传》中有着出色表演的“佟掌柜”同样是军人出身。1990年，闫妮入伍成为一名文艺兵，同年11月进入解放军艺术学院表演系学习，1994年年底，闫妮加入空军政治部电视艺术中心服役至今。</p>
最近因参加《爸爸去哪儿3》走红的“大竣爸爸”林永健曾入伍在广州军区战士话剧团负责剧务工作， 2001年转业到北京，在一次“双拥晚会”上与黄宏 和毛孩 合作演出了小品《西部西瓜》，让林永健萌生了二度入伍的念头，最终成为空政话剧团的一名演员，与妻子周冬齐都是军人。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;韩雪微博截图
许晴、韩雪出身军人家庭</p>
许晴、牛莉、韩雪、陈红都是军人家庭出身。许晴出身外交世家，姥姥是外交官，父母则都是军人。出生在军营的牛莉，父亲是一位职业军人，母亲是射击冠军，从小在军营接受熏陶的牛莉后来也考上了解放军艺术学院。</p>
如今已是娱乐圈“一姐”的范冰冰也曾被传出出身军人之家，爷爷是中国人民解放军海军航空兵的高级将领，父亲是海军航空兵文工团的歌手，转业到地方以后从事文艺工作。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;资料图：李幼斌
李幼斌 ：因敬业精神被特招入伍</p>
曾在《亮剑》《江山》等影视作品中出演多个军人角色的李幼斌走到哪都不忘强调自己是军人。2002年，当时身为长春话剧团演员的李幼斌在八一电影制片厂的电影《惊涛骇浪》中扮演一位全力指挥抗洪的军长，三个月一直坚守在非常艰苦的拍摄现场，敬业精神打动了总政和八一电影制片厂的领导，决定特招李幼斌入伍，圆了他的军人梦。</p>
艺人更需要军人精神</p>
很多人以为，部队所培养出来的那股吃苦的精神，与娱乐圈明星根本无关。其实不然，众所周知，虽然明星大多过着富裕的生活，但拍起戏来可是人人平等，即便是天王天后，只要演戏需要，也得一年四季上刀山下火海，冬天泡得了冰水，夏天穿得住棉衣，甚至有可能面临生命危险。就算是娱乐圈的天王成龙，知天命之年也依然需要吊钢丝，上天入地，摔断腿、摔断肋骨都是家常便饭；章子怡贵为十冠影后，拍《太平轮》时也得在凛冽的冬天泡在又臭又冻的冰水里好几个小时，瑟瑟发抖也必须坚持拍下去。</p>
不管是否受过部队训练，拥有军人般的不畏艰苦的精神与意志，都是好演员必备。而对于曾经当过兵的明星来说，这无疑是一种幸运，军旅生涯的锻炼，让他们在面对艰苦的拍摄条件时比别人多了一份从容，拥有更多机会塑造出更多优秀的角色，可以说，正是部队生活赋予他们的那股韧劲儿，铸就了不少知名艺人今天的成就。</p>
军旅生涯让他们受益匪浅</p>
然而，不管是军旅生涯的磨炼，或是从小受军人作风的影响，都让他们在后来的演艺道路上受益终身。</p>
军人作风影响终身</p>
刘涛在真人秀《花儿与少年》中表现出的雷厉风行，让她“涨粉无数”。在丈夫陷入人生困境时，刘涛的坚忍更是为其赢得贤妻的美名。而这顽强的意志与刘涛曾经的军旅磨炼是分不开的。据悉，在当文艺兵的3年里，刘涛一直是早上5点起床、6点集合，每天还要长跑3公里，挑水、种菜，生活十分艰苦。然而，正是在部队的3年，让她在后来拍戏后变得特别勇敢和有毅力，特别有责任心。</p>
24岁就步入军营的濮存昕拥有强烈的军人情结，濮存昕曾透露，虽然只是一个文艺兵，却照样严格自律。9年的军旅生涯，让濮存昕养成一种踏踏实实、严于律己的精神，对他日后的艺术生涯有着深刻影响。</p>
作为一名军人，韩红有着一股决不放弃的韧劲，也让她拥有强烈的集体荣誉感，韩红除了唱歌之外，还积极投身慈善事业。</p>
部队生活磨炼意志</p>
对于很多明星来说，部队生活让他们养成了顽强的毅力，在拍戏时特别能吃苦。演员殷桃就曾说，自己最怕别人说她娇气：“我不能辜负女军人的头衔。”殷桃回忆，自己第一次去基层部队参加新兵集训，因为当地缺水，每天一个装满的8磅热水瓶就是定额用水，这些水既包括饮用，也包括洗漱用。没想到一天下来大家的水都不够用，战友们推举她去找班长要水，后来班长给解决了，再后来殷桃才知道，她们泡脚用的热水本来是给男战士喝的水。这件事让殷桃成长了许多，觉得自己作为军人，一定不能娇气任性。</p>
军事化的管理和艰苦的舞蹈练习，在很大程度上磨砺了孙俪，退伍后她出演了《玉观音》，为大众熟知。“我们每天早上5点半起来，什么都没干就先得叠被子。叠得慢的话，饭前就甭想洗漱了。那还不算，要是叠得不好，吃完饭就会看到自己的被子从4楼给扔下来。还有牙具、毛巾之类的，只要有一个人放得不整齐，所有人的都会被教官从楼上给扔下来。”谈起军营给予自己的影响，孙俪坦言最重要的就是磨炼了自己的意志：“我都不相信自己能做到的，但真的就做到了。刚来我什么都不行，还经常哭。可没过多少日子，我就成为班里的队列标兵，内务标兵。”</p>
军旅题材演起来很顺手</p>
与此同时，军人生涯所积累的生活经历，也为不少明星日后的创作提供了丰富的灵感。演员梁静一系列的军人角色都曾为观众印象深刻，而这与她曾经的军营生活不无关系。闫妮表示，多年的军旅生涯，让她在出演军人角色时很有优势。</p>
除此之外，梅婷、杨洋在当文艺兵时苦练舞蹈，也为他们日后演戏打下坚实基础。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;资料图：吴亦凡
中新网9月8日电(李萌)近期，吴亦凡为新片剪去头发，以光头造型示人，引发热议。虽然没有了帅气的发型，但气场依旧。</p>
其实，娱乐圈中，明星扮光头造型早不是新鲜事，比如白百何在新片中留光头身披婚纱、范冰冰演尼姑等；也有因个人喜好剃光头的，比如胡军、小沈阳；另外，还有因手误被迫变成光头的，例如女星邓紫棋。</p>
为演戏尝试光头造型</p>
代表明星：吴亦凡、白百合、佟大为、范冰冰、李晨、林志玲、霍建华、吴建豪</p>
吴亦凡出演唐僧 造型帅气</p>
今年8月初，吴亦凡确定加盟电影《西游降魔2》出演唐僧，当月22日，一组吴亦凡参加真人秀录制的照片曝光。照片中，身穿黑色半袖搭配花裤子的他，头戴鸭舌帽，可见以前浓密的头发已消失不见。据悉，这样的造型也是其出演唐僧的一种需要。</p>
虽然变成了光头，但吴亦凡时尚、电影事业依然稳步前进。近日，一组吴亦凡摇滚范儿是写真曝光，有网友评价道：“光头是检验帅哥的唯一标准。”此外，吴亦凡近日还与Dr.Dre等国际重量级明星一起登上美国纽约时代广场，参与一项国际推广活动，成为此次活动中唯一入选的亚太地区明星。</p>
电影方面，6日吴亦凡与冯小刚等出席新片《老炮儿》发布会。现场，冯小刚不仅和吴亦凡打起篮球，还用手摸了摸他的光头。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;资料图：《滚蛋吧！肿瘤君》剧照
白百何剧中剃光头穿婚纱</p>
电影《滚蛋吧！肿瘤君》上映后，因其温馨的故事得到好评。片中，白百何出演的“熊顿”是一位插画家，但年纪轻轻却得了癌症。为了给她加油打气，闺蜜朋友一起剃了光头，穿上婚纱陪其拍艺术照。</p>
其实，这已不是白百何第一次在戏中尝试光头造型，2014年上映的《整容日记》中，白百何就以光头示人，还尝试了各种扮丑模样。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;资料图：佟大为
佟大为演法海 身披袈裟胡子花白</p>
与吴亦凡相同，佟大为在新片中也饰演了一位出家人，只不过演的是法海。8月28日，出席该片探班活动时，佟大为以光头造型亮相，身披袈裟，十分逗趣。据他介绍，法海这个造型看似简单，但却花了3个多小时才完成。</p>
“这个扮相挺有意思。”采访中，佟大为表示虽然稍微夸张一点，但自己刚自拍一张，“还没发给老婆，一会发给她”。他透露自己常常给关悦发一些戏中搞怪造型，但对方都不让孩子看。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;图片来源：台湾“中国时报”
范冰冰变尼姑 网友赞：美到逆天</p>
去年，范冰冰出演的电视剧《武则天》引发热议。剧中，她饰演的武则天尝试了削发为尼的造型，这也是她首次挑战这样的妆容。</p>
剧照中，范冰冰穿着粗衣布鞋，朴素的造型散发出淡然平静的美感，但气质丝毫不输宫中的华服锦衣之美。对此，有网友留言称，“没有见过这么漂亮的尼姑！真是美到逆天”、“冰冰都出家了你还不着急！快接媚娘回宫”、“范爷美到连光头都惊呆了”。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;资料图：《九层妖塔》剧照
李晨变身图书馆馆长</p>
9月初，网上流传一张电影《九层妖塔》海报，主演之一李晨的光头造型随之曝光。</p>
片中，与赵又廷饰演的胡八一对手戏最多的角色，除了姚晨演的Shirley，就是由李晨扮演的王馆长。胡八一从部队复员后，被安排在图书馆工作，当时接待他的正是王馆长。两人从最初的陌生人，发展成了一起吃西餐、骑单车、逛鼓楼、跳爵士的好兄弟。</p>
戏里，李晨带赵又廷走出了昆仑险境留下的阴影，并逐步适应了看似平静的生活。他透露，王馆长将用“温水疗法”治愈赵又廷，他的身份绝不仅仅是老大哥这么简单，还肩负着神秘的使命，而赵又廷正是那个他命中注定要交托重担的人。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 资料图：林志玲。陈秋杏 摄
林志玲假扮尼姑依旧美丽</p>
林志玲曾在电影《天机：富春山居图》里出演“间谍”，为了达成任务，她变换空姐、护士等多个身份，还扮成了尼姑，首次以光头造型亮相。据剧照显示，就算没有头发，林志玲依旧美丽。</p>
2013年6月，在深圳出席该片发布会时，林志玲坦言很喜欢自己光头的造型，不仅突破了自己的固有形象，安静、内敛的表演方式也让自己收获良多。对于被问是否担心会破坏“完美女神”的大众形象，林志玲笑言，“这部影片的‘神’是刘德华，我自己是志玲姐姐”。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 《镖门》剧照(图片来源：深圳特区报)
霍建华为角色主动剃光头</p>
昨日，由霍建华、赵丽颖等主演的电视剧《花千骨》正式收官，虽然在剧中，霍建华长发白衣的仙人形象获得观众好评，但此前，他也曾尝试过光头的造型。</p>
2014年1月，由其主演的电视剧《镖门》在北京热拍，剧中，霍建华首次剃光头演出。制片人周劲翔回忆说：“当时霍建华看完剧本，第一次与剧组见面时，他还没签约。他戴着帽子进屋，走到我们面前把帽子一摘，大家都惊了――原来他是剃好光头过来的，他是想让我们放心，这部剧他演定了。”</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;图片来源：台湾“中国时报”
吴建豪变“大胡子怪”</p>
吴建豪曾凭借《流星花园》中花美男“美作”一角出道，近年来却开始走起“大叔范”。9月初，他在网上晒出留胡子近照。粉丝看后一惊，直呼“怎么变大胡怪了”。</p>
照片中，吴建豪变成光头，留着大胡子，网友看到后纷纷表示，吴建豪昔日花美男外貌已不复见。不久后他又晒出与林书豪合拍的影片截图，自称是“因戏牺牲”，大胡子形象是暂时性的。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 图片来源：林志颖微博
生活中尝试光头造型</p>
代表人物：林志颖、小沈阳、胡军、邓紫棋、梁朝伟</p>
林志颖光头造型遭质疑</p>
2013年8月，为庆祝微博粉丝突破2100万，林志颖晒出一张光头照，站在一群人当中十分抢镜。平日里皮肤白皙的他，在这张照片中变得黝黑，赤裸上身，肌肉紧实，更显男性成熟魅力。</p>
对此，有网友留言称，“这到底是P的图，还是真实的好身材”、“你是要去演十八铜人了吗”、“感觉是修图的吧”。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 小沈阳微博截图
小沈阳为爱妻庆生：我爱你</p>
今年3月，小沈阳在微博晒出一组与妻子沈春阳的合照，为对方庆生，还留言道：“这半辈子欠你太多，就这样牵手下去！老婆生日快乐！我爱你。”照片中，剃了光头的小沈阳，正亲吻妻子沈春阳脸颊，另外，夫妻俩还做出各种搞怪表情，幸福甜蜜。</p>
照片曝光后，网友纷纷点赞留言，“我终于知道啥是真爱了”，“不要在乎别人嘴里说什么，做好自己就行，祝你们幸福哥哥嫂子”。</p>

胡军与林青霞合影卖萌</p>
去年11月3日是林青霞的60岁生日。当月4日，胡军在微博晒出和寿星林青霞的亲密合影，引发网友关注。</p>
照片中，留着光头的胡军和林青霞开心合影，还比出剪刀手“卖萌”，称“美丽的青霞生日快乐”。网友看后纷纷送上祝福，“军哥最帅”、“女侠，生日快乐”。</p>

邓紫棋手滑剃掉半边头发</p>
去年，邓紫棋因参加《我是歌手》第二季备受关注。录制节目期间，她也曾分享被自己剪成光头的糗事。</p>
据透露，在拍摄MV时，她要对着镜头将自己右边的头发剃掉一部分，可是因为必须面对镜头，没有镜子可以看，结果手一滑，将自己整个右边头发全部剃掉了，导致接下来的一年中，她一直是“阴阳头”造型，令邓紫棋叫苦连天。</p>

梁朝伟脱发 疑因心理压力大</p>
2014年11月，梁朝伟曾多次以三分头造型亮相，与此前斯文形象相比，少了几分忧郁，多了几分沉稳。然而，他的发线明显后移，额上还向后空出一块倒梯形的几何秃面，似已出现脱发迹象，心理压力可见一斑。</p>
据悉，当年8月初，梁朝伟就已经剃发，在宣传《一代宗师》时，发线凹处就十分显眼，传其情绪、心理压力可能是主因，后也有香港媒体报道称，梁朝伟还曾为情绪问题到中环等地求助中医。</p>
因华为在新浪发布名为“双眼见花千骨”的微博话题， 被演员赵丽颖、霍建华将华为和新浪告上法庭，认为两被告侵害肖像权、姓名权，要求立即停止侵权，在新浪微博首页及全国公开发行报纸登报致歉，赔偿精神损害抚慰金等损失167万元。法晚记者上午获悉，海淀法院已经受理了此案。</p>
赵丽颖、原告霍建华是电视连续剧《花千骨》的主要演员。今年8月，两人得知在北京微梦创科网络技术有限公司经营的网站中，华为官方微博“华为荣耀”发布了名为“双眼见花千骨”的微博话题，该话题擅用两人肖像等，对华为手机进行商业推广宣传。两人认为，该行为会误导浏览者认为原告与华为存在某种商业合作关系，涉嫌侵犯原告的肖像权和姓名权。(记者 王子薇)</p>
中新网北京9月7日电(记者 张曦) 主演汤姆・克鲁斯(阿汤哥)今天现身北京，宣传新片《碟中谍5:神秘国度》(以下简称《碟中谍5》)，他全程亮相不到15分钟，除了用中文讲“你好”、“谢谢”外，还直言拍摄该系列电影已近20年，与戏中角色感情非常深。
《碟中谍5》讲述了阿汤哥饰演的伊桑・亨特与其团队一起遭遇到一个名为“神秘国度”的组织追杀的故事。</p>
作为中国之行的最后一站，阿汤哥今天与导演克里斯托夫・迈考利、女主角丽贝卡・弗格森一同亮相，三人一露面就引发全场尖叫，阿汤哥也不断向影迷挥手致意。</p>
“这次整个中国之行都非常好，影迷们都非常热情。”阿汤哥透露自己明天就要回好莱坞继续拍摄新片。对于《碟中谍5》，他直言这次充斥了很多元素，“不仅有悬疑、动作、搞笑，还能看到团队作战、情节逆转，相信大家一定会喜欢”。</p>
从1996年第一部《碟中谍》面世以来，阿汤哥饰演的伊桑・亨特已经伴随影迷近20年，他今天也感慨良深地说：“这是一部经典的动作电影，很高兴能出演，我与角色的感情也非常深。”</p>
此次《碟中谍5》中，阿汤哥继续挑战高难度动作戏，其中爬飞机等场景都让影迷捏了把汗，他坦言这些危险戏份都是自己亲自上阵，并未使用替身。</p>
丽贝卡・弗格森不舍离开中国</p>
谈到危险戏份，女主角丽贝卡・弗格森也滔滔不绝，她直言这是自己出演的第一部动作电影，而且不乏危险戏份，“开拍前我接受了四到五周的训练，拍摄中我也一直和阿汤哥配合训练，所以很享受这个过程”。</p>
