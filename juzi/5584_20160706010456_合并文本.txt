目前，案件正在进一步审理中。丁女士的刑事附带民事诉讼代理律师苏先生表示，这件事每次提及，对丁女士都是一种伤害。“她希望这件事赶快过去，让施暴者及犯罪者都受到法律的惩罚。”

声明还提到，蒋劲夫提解约一事，有第三方以高利益诱惑使艺人心态发生变化。</p>
蒋劲夫2010年参加《快乐大本营》校花校草系列节目而受到关注，2012年，还在上海戏剧学院读书的他因主演唐人的《轩辕剑之天之痕》而受到广泛的关注，他也因此签入唐人，并接连出演了《步步惊情》《秦时明月》《青丘狐传说》等作品。唐人近期正在拍摄的《仙剑奇侠传5》本也预定由蒋劲夫主演，但随后却又传出蒋劲夫辞演的消息。

曾出演过《一江春水向东流》、《万家灯火》的著名演员高正于9月6日去世，享年93岁。高正于1922年出生，1946年考入昆仑影业公司，并逐渐成为昆仑影业的当红小生。除了出演电影，高正也担任导演，执导过《平鹰坟》、《桔园情》等影视作品。</p>
曾与高正合作过的演员佟瑞欣回忆老爷子，称其温文尔雅，“每次拍摄情感戏都会看到他或悲伤的眼泪或喜悦的笑容”。上海电影家协会副主席石川在微博上怀念老爷子，他写道：“他(高正)的形象与经典同在：《万家灯火》里的小赵、《八千里路云和月》里的堂哥、《林则徐》里的道光皇帝！老爷子90岁时还腰杆倍儿挺，看着还像小鲜肉！一路走好！永远青春！”(杨莲洁)</p>

近日，不少明星为纪念反法西斯战争胜利70周年，纷纷晒出敬礼照或者军装照，向抗战老兵致敬。事实上，娱乐圈里当过军人或出生军人家庭的明星不在少数。他们或多或少都经历过军旅生涯的锻炼，练就了顽强拼搏的军人性格，为日后进入娱乐圈打下坚实的基础。</p>
孙俪、刘涛都曾是文艺兵</p>
5岁时孙俪便进入少年宫学习舞蹈，初中毕业，上海警卫区文工团招收新兵，孙俪幸运地当上了一名文艺兵。董洁 10岁由战士歌舞团在大连特招入伍，后被送入北京的解放军艺术学院学了5年舞蹈。1995年底正式到战士歌舞团做舞蹈演员，后来因为和谢霆锋 在春晚的合作而被张艺谋 相中主演《幸福时光》成名。歌唱组合“凤凰传奇”2009年进入二炮文工团，那首红遍大江南北的《中国我爱你》就是凤凰传奇特意为2009年建国60周年大庆而献声的歌曲。</p>
演员刘涛1993年被选入南京军区集团军文工团，在那里当了3年的文艺兵；梅婷也曾有过8年的军旅生涯；而在年轻人中颇受欢迎的“小鲜肉”杨洋，同样毕业于解放军艺术学院舞蹈系，是一名文艺兵。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 资料图：韩红。 中新社发 陈文 摄
韩红、闫妮目前仍是军人</p>
也有不少明星目前仍是现役军人。除了知名歌手韩红之外，曾在《武林外传》中有着出色表演的“佟掌柜”同样是军人出身。1990年，闫妮入伍成为一名文艺兵，同年11月进入解放军艺术学院表演系学习，1994年年底，闫妮加入空军政治部电视艺术中心服役至今。</p>
最近因参加《爸爸去哪儿3》走红的“大竣爸爸”林永健曾入伍在广州军区战士话剧团负责剧务工作， 2001年转业到北京，在一次“双拥晚会”上与黄宏 和毛孩 合作演出了小品《西部西瓜》，让林永健萌生了二度入伍的念头，最终成为空政话剧团的一名演员，与妻子周冬齐都是军人。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;韩雪微博截图
许晴、韩雪出身军人家庭</p>
许晴、牛莉、韩雪、陈红都是军人家庭出身。许晴出身外交世家，姥姥是外交官，父母则都是军人。出生在军营的牛莉，父亲是一位职业军人，母亲是射击冠军，从小在军营接受熏陶的牛莉后来也考上了解放军艺术学院。</p>
如今已是娱乐圈“一姐”的范冰冰也曾被传出出身军人之家，爷爷是中国人民解放军海军航空兵的高级将领，父亲是海军航空兵文工团的歌手，转业到地方以后从事文艺工作。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;资料图：李幼斌
李幼斌 ：因敬业精神被特招入伍</p>
曾在《亮剑》《江山》等影视作品中出演多个军人角色的李幼斌走到哪都不忘强调自己是军人。2002年，当时身为长春话剧团演员的李幼斌在八一电影制片厂的电影《惊涛骇浪》中扮演一位全力指挥抗洪的军长，三个月一直坚守在非常艰苦的拍摄现场，敬业精神打动了总政和八一电影制片厂的领导，决定特招李幼斌入伍，圆了他的军人梦。</p>
艺人更需要军人精神</p>
很多人以为，部队所培养出来的那股吃苦的精神，与娱乐圈明星根本无关。其实不然，众所周知，虽然明星大多过着富裕的生活，但拍起戏来可是人人平等，即便是天王天后，只要演戏需要，也得一年四季上刀山下火海，冬天泡得了冰水，夏天穿得住棉衣，甚至有可能面临生命危险。就算是娱乐圈的天王成龙，知天命之年也依然需要吊钢丝，上天入地，摔断腿、摔断肋骨都是家常便饭；章子怡贵为十冠影后，拍《太平轮》时也得在凛冽的冬天泡在又臭又冻的冰水里好几个小时，瑟瑟发抖也必须坚持拍下去。</p>
不管是否受过部队训练，拥有军人般的不畏艰苦的精神与意志，都是好演员必备。而对于曾经当过兵的明星来说，这无疑是一种幸运，军旅生涯的锻炼，让他们在面对艰苦的拍摄条件时比别人多了一份从容，拥有更多机会塑造出更多优秀的角色，可以说，正是部队生活赋予他们的那股韧劲儿，铸就了不少知名艺人今天的成就。</p>
军旅生涯让他们受益匪浅</p>
然而，不管是军旅生涯的磨炼，或是从小受军人作风的影响，都让他们在后来的演艺道路上受益终身。</p>
军人作风影响终身</p>
刘涛在真人秀《花儿与少年》中表现出的雷厉风行，让她“涨粉无数”。在丈夫陷入人生困境时，刘涛的坚忍更是为其赢得贤妻的美名。而这顽强的意志与刘涛曾经的军旅磨炼是分不开的。据悉，在当文艺兵的3年里，刘涛一直是早上5点起床、6点集合，每天还要长跑3公里，挑水、种菜，生活十分艰苦。然而，正是在部队的3年，让她在后来拍戏后变得特别勇敢和有毅力，特别有责任心。</p>
24岁就步入军营的濮存昕拥有强烈的军人情结，濮存昕曾透露，虽然只是一个文艺兵，却照样严格自律。9年的军旅生涯，让濮存昕养成一种踏踏实实、严于律己的精神，对他日后的艺术生涯有着深刻影响。</p>
作为一名军人，韩红有着一股决不放弃的韧劲，也让她拥有强烈的集体荣誉感，韩红除了唱歌之外，还积极投身慈善事业。</p>
部队生活磨炼意志</p>
对于很多明星来说，部队生活让他们养成了顽强的毅力，在拍戏时特别能吃苦。演员殷桃就曾说，自己最怕别人说她娇气：“我不能辜负女军人的头衔。”殷桃回忆，自己第一次去基层部队参加新兵集训，因为当地缺水，每天一个装满的8磅热水瓶就是定额用水，这些水既包括饮用，也包括洗漱用。没想到一天下来大家的水都不够用，战友们推举她去找班长要水，后来班长给解决了，再后来殷桃才知道，她们泡脚用的热水本来是给男战士喝的水。这件事让殷桃成长了许多，觉得自己作为军人，一定不能娇气任性。</p>
军事化的管理和艰苦的舞蹈练习，在很大程度上磨砺了孙俪，退伍后她出演了《玉观音》，为大众熟知。“我们每天早上5点半起来，什么都没干就先得叠被子。叠得慢的话，饭前就甭想洗漱了。那还不算，要是叠得不好，吃完饭就会看到自己的被子从4楼给扔下来。还有牙具、毛巾之类的，只要有一个人放得不整齐，所有人的都会被教官从楼上给扔下来。”谈起军营给予自己的影响，孙俪坦言最重要的就是磨炼了自己的意志：“我都不相信自己能做到的，但真的就做到了。刚来我什么都不行，还经常哭。可没过多少日子，我就成为班里的队列标兵，内务标兵。”</p>
军旅题材演起来很顺手</p>
与此同时，军人生涯所积累的生活经历，也为不少明星日后的创作提供了丰富的灵感。演员梁静一系列的军人角色都曾为观众印象深刻，而这与她曾经的军营生活不无关系。闫妮表示，多年的军旅生涯，让她在出演军人角色时很有优势。</p>
除此之外，梅婷、杨洋在当文艺兵时苦练舞蹈，也为他们日后演戏打下坚实基础。</p>

其实，娱乐圈中，明星扮光头造型早不是新鲜事，比如白百何在新片中留光头身披婚纱、范冰冰演尼姑等；也有因个人喜好剃光头的，比如胡军、小沈阳；另外，还有因手误被迫变成光头的，例如女星邓紫棋。</p>
为演戏尝试光头造型</p>
